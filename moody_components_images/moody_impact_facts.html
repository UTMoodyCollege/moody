Moody Impact Facts¶

The Moody Impact Facts component provides repeatable units (max of 4) each with headline and subheadline fields. The headline field displays in a extra-large bold font for emphasis. There is a style selector where you can choose “Orange headline with charcoal subheadline” or “Charcoal headline with orange subheadline”. There is no limit on the number of facts you can add and there is a selector where you can choose to display two, three or four impact facts per row.

Caution

If you add multiple units always check them at all viewport widths to verify there is adequate space between the units. Because the headline font is so large they have a tendency to outgrow their container if too much text is used.
